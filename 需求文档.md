这是一个视频合成工具，使用 Python 开发。

## 功能特点
- 支持输入多段视频素材
- 支持输出视频音频
- 支持输入字幕文件

## 输入

输入一个有序的素材数组，每个对象包含以下字段：

```json
[
    {
        "order": 1,
        "video_url": "https://example.com/video1.mp4",
        "audio_url": "https://example.com/audio1.mp3",
        "subtitle_url": "https://example.com/subtitle1.srt",
      
    },
    {
        "order": 2,
        "video_url": "https://example.com/video2.mp4",
        "audio_url": null,
        "subtitle_url": "https://example.com/subtitle2.srt"
    }
]
```

### 字段说明
- `order` (必填): 整数，视频片段的播放顺序
- `video_url` (必填): 字符串，视频素材的URL地址
- `audio_url` (可选): 字符串或null，音频文件URL。如果提供，将替换视频原有音频
- `subtitle_url` (可选): 字符串或null，字幕文件URL（支持SRT格式），字幕时间轴针对单个片段

### 处理规则
- 按 `order` 字段顺序依次合成视频片段
- 如果提供 `audio_url`，则替换该片段视频的原有音频
- 如果提供 `subtitle_url`，则将字幕叠加到该片段视频上（字幕时间从该片段开始计算）
- 片段之间直接拼接，不做裁剪或转场效果

## 输出
- 输出视频文件（MP4 格式）
- 将接口请求提供的视频素材进行合成，并输出最终视频文件。
